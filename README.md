## 安装 Skill（给 OpenClaw 用户）

将此 Skill 安装到你自己的 OpenClaw 环境：

```bash
# 方法一：从 GitHub 直接安装
clawhub install https://raw.githubusercontent.com/Liiin96/xhs-knowledge-hub/main/xhs-knowledge-hub.skill

# 方法二：下载后本地安装
wget https://raw.githubusercontent.com/Liiin96/xhs-knowledge-hub/main/xhs-knowledge-hub.skill
clawhub install ./xhs-knowledge-hub.skill
```

安装后，直接对话：
> 「帮我搭建一个 XXX 行业知识库」
> 「更新知识库，新增一篇文档到行业范式」
